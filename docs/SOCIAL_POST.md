# Prayer Times Chrome Extension — Social Post

---

**I kept missing prayers at work. So I built this.**

8 hours a day in Chrome. Meetings, coding, tabs everywhere. I'd look up and realize — oh, Maghrib was 20 minutes ago.

Started as a tiny extension just for Cairo. Then thought, why not make it work everywhere?

**What I actually use daily:**

- **Browser badge countdown** — one glance, no clicks
- **Pop-out timer** — I keep it on my second screen like a dashboard widget
- Works offline, supports 40+ countries, Arabic + English

**Try it in 2 minutes:**
1. Download ZIP from GitHub
2. `chrome://extensions/` → Developer mode ON → Load unpacked → pick the `dist/` folder

That's it. No setup, no accounts.

If it's not just me who finds this useful, I'll publish it on the Chrome Web Store.
