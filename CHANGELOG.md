# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-27

### Added
- Initial release with Manifest V3 Chrome extension
- Live prayer times via Aladhan.com API
- Offline fallback using Adhan JS local calculation
- Next prayer countdown in popup and extended floating window
- Background auto-sync at 3:00 AM
- Toolbar badge showing minutes to next prayer
- Desktop notifications for prayer times
- Options page: city selector, custom coordinates, calculation method, madhab, time format, language, theme
- Full Arabic localization with RTL support
- Light/Dark/System themes
- GitHub Actions CI/CD workflow for automated builds and releases
